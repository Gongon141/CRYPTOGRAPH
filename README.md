# Шифр нигилистов (Nihilist Cipher) на C++

Реализация исторического блочного шифра, которым пользовались русские революционеры-нигилисты в конце XIX века.  
Шифр объединяет квадрат Полибия 5×5 (латиница, I/J объединены) с числовым гаммированием (сложением координат).

## 📖 Как работает алгоритм

1. **Квадрат Полибия** строится на основе ключа-таблицы. Ключ записывается без повторений, затем дополняется оставшимися буквами латиницы (A-Z без J). Строки и столбцы нумеруются от 1 до 5, координаты записываются двузначным числом `строка*10 + столбец`.
2. **Гамма-ключ** (второй секретный ключ) повторяется до длины открытого текста.
3. **Шифрование** – для каждой позиции координаты буквы текста и координаты буквы гаммы **суммируются**. Полученные числа (от 22 до 110) образуют шифротекст.
4. **Дешифрование** – из чисел шифротекста вычитаются координаты соответствующих букв гамма-ключа. Результат (11..55) разбивается на строку и столбец, по таблице восстанавливается буква.

## 🔧 Функциональность программы

- Построение таблицы Полибия 5×5 по ключу.
- Шифрование любого текста на латинице (пробелы и небуквенные символы удаляются).
- Дешифрование из последовательности чисел.
- Интерактивное меню с выбором операции.
- Обработка ошибок (неправильные координаты, пустой ввод и т.д.).
