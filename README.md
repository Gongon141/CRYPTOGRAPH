# Классические шифры на C++: Нигилистов и Плейфера

Реализация двух исторических блочных шифров на C++ с интерактивным консольным интерфейсом:

- **Шифр нигилистов** (квадрат Полибия + числовое гаммирование)
- **Шифр Плейфера** (Playfair cipher) на матрице 5×5

Оба алгоритма работают с латинским алфавитом (буквы I и J объединены). Ключи и тексты регистронезависимы, пробелы и небуквенные символы игнорируются.

## 📖 Алгоритмы

### 1. Шифр нигилистов (Nihilist Cipher)

Основан на квадрате Полибия 5×5. Требуется **два ключа**:

- **Ключ таблицы** — заполняет квадрат.
- **Гамма-ключ** — накладывается на координаты открытого текста.

**Шифрование**:  
Каждая буква открытого текста и соответствующая буква гаммы заменяются на двузначное число (строка·10 + столбец из квадрата), затем числа суммируются. Результат — последовательность чисел (22…110).

**Дешифрование**:  
Из чисел шифротекста вычитаются координаты букв гаммы, полученные координаты (11…55) переводятся обратно в буквы по тому же квадрату.

### 2. Шифр Плейфера (Playfair Cipher)

Классический биграммный шифр. Использует **один ключ** для построения матрицы 5×5 (без J, I=J). Открытый текст разбивается на пары букв (биграммы). Правила преобразования биграмм:

- Если буквы в одной строке — каждая заменяется на соседнюю справа (циклично).
- Если в одном столбце — на соседнюю снизу.
- Иначе — берутся противоположные углы прямоугольника.

При необходимости в открытый текст вставляется `X` (для разделения повторяющихся букв в паре и дополнения нечётной длины).

## 🧩 Функциональность программы

- Интерактивный ввод ключей и текста.
- Меню выбора операции: шифрование, дешифрование, смена алгоритма.
- Поддержка двух алгоритмов из одного интерфейса.
- Очистка текста от пробелов и неалфавитных символов.
- Обработка ошибок (некорректный ввод, пустой текст, невозможные координаты).
