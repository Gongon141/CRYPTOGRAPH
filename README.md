# Классические шифры на C++: XOR и Атбаш

Реализация двух простых, но показательных шифров на C++ с интерактивным консольным интерфейсом:

- **Шифр Атбаш** (Atbash cipher) — шифрование подстановкой с фиксированным зеркальным алфавитом.
- **Шифр XOR** (XOR cipher) — симметричное поточное шифрование с побайтовым ключом.

Оба алгоритма работают с латинским алфавитом (регистронезависимо). Пробелы и небуквенные символы игнорируются.

## 📖 Алгоритмы

### 1. Шифр Атбаш (Atbash Cipher)

Классический шифр подстановки, где каждая буква заменяется на свою "зеркальную" в алфавите (A ↔ Z, B ↔ Y, C ↔ X и т.д.).

**Шифрование и дешифрование** выполняются одинаково (инволютивный алгоритм):
- Для символа вычисляется его позиция в алфавите (0–25).
- Новая позиция = 25 - текущая позиция.
- Буква заменяется на соответствующую в этой позиции.

**Пример**: `HELLO` → `SVOOL`, `WORLD` → `DLIOW`

### 2. Шифр XOR (XOR Cipher)

Симметричный поточный шифр, где каждый байт открытого текста побитово складывается по модулю 2 (XOR) с байтом ключа. При повторном применении того же ключа текст восстанавливается.

**Шифрование/дешифрование**:
- Текст преобразуется в последовательность байт (ASCII).
- Каждый байт текста XOR'ится с соответствующим байтом ключа.
- Если ключ короче текста, он циклически повторяется.
- Результат выводится в шестнадцатеричном виде (для наглядности).

**Пример** (ключ `KEY`):
- `HELLO` XOR `KEYKE` → `3A 0C 14 1C 0A` (шестнадцатеричный вывод)

## 🧩 Функциональность программы

- Интерактивный ввод ключа (для XOR) и текста.
- Меню выбора операции: шифрование, дешифрование, смена алгоритма.
- Поддержка двух алгоритмов из одного интерфейса.
- Очистка текста от пробелов и неалфавитных символов (для Атбаш).
- Для XOR — отображение результата в hex-формате.
- Обработка ошибок (некорректный ввод, пустой текст).
